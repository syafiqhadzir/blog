# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly.

### Contact

- **Email:** [syafiqhadzir@live.com.my](mailto:syafiqhadzir@live.com.my)
- **Response Time:** Within 48 hours

### Guidelines

1. **Do not** publicly disclose the vulnerability until it has been addressed
2. Provide detailed information about the vulnerability
3. Include steps to reproduce the issue if possible

### Scope

This security policy applies to:

- The blog website ([blog.syafiqhadzir.dev](https://blog.syafiqhadzir.dev))
- This GitHub repository

### Acknowledgments

We appreciate responsible disclosure and will acknowledge security researchers
who help improve our security.

## Security Measures

This site implements:

- HTTPS enforcement via GitHub Pages
- Content Security Policy compatible with AMP
- No user data collection or storage
- Static site generation (no server-side vulnerabilities)
